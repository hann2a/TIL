```python
"""
큐 연습문제 “마이쮸를 나눠주자”

**문제**

담임 선생님이 학생들에게 마이쮸를 나눠주려고 합니다. 마이쮸를 나눠주는 규칙이 조금 독특하여, 
지막 마이쮸를 받아가는 학생이 누구일지 궁금해졌습니다. 큐(Queue) 자료구조를 이용하여 이 과정을 시뮬레이션하고, 
마지막 마이쮸를 받는 학생의 번호를 찾는 프로그램을 작성하세요.

**진행 규칙**

1. 초기 마이쮸는 총 20개입니다.
2. 시뮬레이션은 1번 학생이 1개의 마이쮸를 받기 위해 줄을 서면서 시작됩니다.
3. 줄의 맨 앞`에 있는 학생이 나와서 자신이 받기로 한 개수만큼 마이쮸를 받습니다.
4. 마이쮸를 받은 학생은, 다음에 받을 마이쮸 개수를 현재 받은 개수보다 1개 더 늘려서 다시 줄의 맨 뒤로 갑니다.
5. 기존 학생이 줄을 선 직후, 아직 줄을 선 적 없는 새로운 번호의 학생이 1개의 마이쮸를 받기 위해 줄의 맨 뒤에 이어서 섭니다.
6. 마이쮸를 나눠주다가 남은 개수가 학생이 받으려는 개수보다 부족하면, 그 학생이 남은 마이쮸를 모두 가져가면서 시뮬레이션이 종료됩니다.

**목표**

총 20개의 마이쮸가 모두 소진되었을 때, 마지막 마이쮸를 받아가는 학생은 몇 번 학생인지 찾는 프로그램을 작성하세요.
"""

# 전체 마이쮸의 개수
total_candy = 20

# 학생들의 대기열
# 큐에는 (학생 번호, 다음에 받을 마이쮸 개수) 튜플을 저장합니다.
queue = []

# 시뮬레이션 시작: 1번 학생이 1개를 받기 위해 줄을 섭니다.
queue.append((1, 1))

# 마지막으로 마이쮸를 받은 학생의 번호를 저장할 변수
last_person = None

# 다음에 새로 줄을 설 학생의 번호
next_person = 2

# --- 시뮬레이션 시작 ---
"""
<시뮬레이션 과정>
큐에서 꺼냄
마이쮸 나눠줌 > 누가 받았는지 기록 > 다시 줄서게함 > 새로운 사람 줄 서게함
> 다시 pop 해서 큐에서 꺼냄 
"""
# 마이쮸가 남아있는 동안 반복 
while total_candy > 0:
    # 1. 큐의 맨 앞에서 대기중인 학생을 꺼냄 
    person, count = queue.pop(0)

    # 2. 마이쮸를 나눠줌 
    # 만약 남은 마이쮸 < 학생이 요구하는 양 -> 이 학생이 마지막 
    if total_candy <= count:
        last_person = person
        break 

    # 마지막이 아니라면, 마이쮸를 나눠주고, 총 개수에서 차감 
    total_candy -= count 

    # 마지막 수령자 기록 
    last_person = person

    # 마이쮸 받은 학생이 다시 줄을 섬 
    # 다음에 받을 개수는 현재 받은 개수 + 1
    queue.append((person, count +1))

    # 새로운 학생이 줄을 섬 
    queue.append((next_person, 1))

    # 새로운 학생도 받았으므로 다음 사람으로 넘김 
    next_person += 1

# 마지막으로 마이쮸를 받은 사람을 출력
print(f"마지막 마이쮸를 가져간 사람은 {last_person}번 학생입니다.")
```
