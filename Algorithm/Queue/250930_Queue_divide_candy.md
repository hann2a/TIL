```python
"""
큐 연습문제 “마이쮸를 나눠주자”

**문제**

담임 선생님이 학생들에게 마이쮸를 나눠주려고 합니다. 마이쮸를 나눠주는 규칙이 조금 독특하여, 
지막 마이쮸를 받아가는 학생이 누구일지 궁금해졌습니다. 큐(Queue) 자료구조를 이용하여 이 과정을 시뮬레이션하고, 
마지막 마이쮸를 받는 학생의 번호를 찾는 프로그램을 작성하세요.

**진행 규칙**

1. 초기 마이쮸는 총 20개입니다.
2. 시뮬레이션은 1번 학생이 1개의 마이쮸를 받기 위해 줄을 서면서 시작됩니다.
3. 줄의 맨 앞`에 있는 학생이 나와서 자신이 받기로 한 개수만큼 마이쮸를 받습니다.
4. 마이쮸를 받은 학생은, 다음에 받을 마이쮸 개수를 현재 받은 개수보다 1개 더 늘려서 다시 줄의 맨 뒤로 갑니다.
5. 기존 학생이 줄을 선 직후, 아직 줄을 선 적 없는 새로운 번호의 학생이 1개의 마이쮸를 받기 위해 줄의 맨 뒤에 이어서 섭니다.
6. 마이쮸를 나눠주다가 남은 개수가 학생이 받으려는 개수보다 부족하면, 그 학생이 남은 마이쮸를 모두 가져가면서 시뮬레이션이 종료됩니다.

**목표**

총 20개의 마이쮸가 모두 소진되었을 때, 마지막 마이쮸를 받아가는 학생은 몇 번 학생인지 찾는 프로그램을 작성하세요.
"""
total_candy = 20 

queue = []

queue.append((1, 1))

# 정답 변수 
last_person = None

# 다음 번호로 넘기기 위해 필요한 변수 
next_person = 2 

while total_candy > 0: # total_candy가 남아있을 때 !
    num, count = queue.pop(0)
 
    if total_candy - count <= 0: # 0 포함해야됨 ! 
        last_person = num 
        break 

    total_candy -= count 
    last_person = num 

    queue.append((num, count+1))
    queue.append((next_person, 1))

    next_person += 1

# 마지막으로 마이쮸를 받은 사람을 출력
print(f"마지막 마이쮸를 가져간 사람은 {last_person}번 학생입니다.")
```
